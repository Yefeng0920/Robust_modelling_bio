# Robust approaches to modelling fragile conservation biology: flexible solutions to non-normal, hierarchical, temporal, spatial, and phylogenetic data

## Overview

This is a repository containing the reproducible code for the illustrative examples from 

> Yefeng Yang, Jinming Pan, David Warton, Shinichi Nakagawa. Robust approaches to modelling fragile conservation biology: flexible solutions to non-normal, hierarchical, temporal, spatial, and phylogenetic data. 2026.


### `Script`

The `Script` folder includes `Example.Rmd` which contains the R scripts used to reproduce all examples detecting autocorrelation, heteroscedasticity, and outliers. It illustrates how to fit a model accounting for a structured trend and residual dependence. `Example.html` shows interactive results.
